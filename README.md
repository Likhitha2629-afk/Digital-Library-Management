# 📚 Digital Library Management System

A simple and responsive Digital Library Management System built using HTML, CSS, and JavaScript. This project allows users to log in, manage books, search books, and issue or return books through an interactive interface.

## 🚀 Features

- User Login Authentication
- Add New Books
- Search Books
- View Available Books
- Issue Books
- Return Books
- Delete Books
- Logout Functionality
- Responsive User Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📖 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. Login using the demo credentials.
5. Manage the library by adding, searching, issuing, returning, or deleting books.
